# TODO-App
A todo web application that helps you to add your tasks with some features.

<b>#How to Use:</b>
1.Get NodeJs on your System.
2.Clone this repository .
3.Open in terminal/cmd
4.Do npm install
5.Go to Web Browser and enter https://localhost:3000
